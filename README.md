# 🏠 HouseHunt - House Rent Management System

## 📌 Project Overview
HouseHunt is a MERN Stack web application that helps users search, view, and book rental houses online. Owners can add and manage house listings, while users can browse available properties and make bookings through a user-friendly interface.

## 🚀 Features
- User Registration & Login (JWT Authentication)
- Browse Available Houses
- Search Houses
- View House Details
- Book Houses
- Owner/Admin House Management
- Responsive User Interface

## 🛠️ Tech Stack
### Frontend
- React.js
- Vite
- CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
